Overview of the Project


The game asks 16 multiple-choice questions of increasing difficulty.
Every correct response yields higher prize money, while certain checkpoints guarantee winnings.


A single wrong answer ends the game with the player taking home the last guaranteed amount.



 Features
- 16 general knowledge questions
- Multiple-choice format with 4 options per question
- Increasing prize money for each correct answer
- Guaranteed reward milestones at ₹10,000, ₹3,20,000, and ₹1,00,00,000


- Game ends on wrong answer

 Full scoring and final result display

 Steps to Install & Run the Project


1. Install Python (version 3.6 or above recommended)

2. Copy the complete game script into a .py file


Example: knowledge_vault.py




3. Open terminal/command prompt and change the location to your file's location

4. Run the game using:

python knowledge_vault.py

5. Follow the on-screen instructions, entering your answers using number keys (1-4)
