Problem Statement:

The project implements a quiz-based game showing users multiple-choice questions. The
objective is to test the user's general knowledge while awarding increasing prize amounts for each
correct answer. The game is over if the user gives an incorrect answer.

Project Scope:

The project encompasses the creation of a question bank, displaying questions with multiple choices.
options, taking user input, evaluating correctness, updating prize values, and determining game
termination conditions.

Target Users:

The target users include students, quiz enthusiasts, and people who either want to test or improve their
General knowledge in an interactive and gamified manner.

High-Level Features:

Multiple-choice question system Increasing reward levels according to the difficulty level of a question
Immediate feedback for correct or wrong answers; automatic game termination upon incorrect
- Structured prize-winning progression system
